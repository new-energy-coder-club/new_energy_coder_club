
# Docs

> 文档中心，存储所有项目文档和指南

## 📋 目录说明

**用途**：提供统一的文档入口，支持知识管理和传承

**访问级别**：important

**最大深度**：2 层

**命名规范**：`docs/{doc-type}`

## 📁 子目录

- `guides/` - 使用指南和教程
- `api/` - API文档和接口说明
- `tutorials/` - 详细教程和示例
- `references/` - 参考文档和规范
- `archive/` - 历史文档归档


## 📖 使用示例

- `docs/guides/project-setup`
- `docs/tutorials/embedded-development`
- `docs/api/robotics-framework`


## 📝 文件要求

### 必需文件
- `README.md`
- `index.md`

### 可选文件
- `CONTRIBUTING.md`
- `FAQ.md`


## 🚀 快速开始

1. 在相应子目录下创建项目文件夹
2. 添加项目 README.md 文件
3. 按照命名规范组织文件结构
4. 更新上级目录的索引文件


# Competitions

> 竞赛项目目录，按年份和竞赛类型组织

## 📋 目录说明

**用途**：存储所有竞赛相关项目，实现快速访问

**访问级别**：core

**最大深度**：3 层

**命名规范**：`competitions/{year}/{competition-type}`

## 📁 子目录

- `2024/` - 2024年度竞赛项目
- `2025/` - 2025年度竞赛项目
- `archive/` - 历史竞赛项目归档


## 📖 使用示例

- `competitions/2024/electronics-competition`
- `competitions/2024/energy-saving-competition`
- `competitions/2025/robotics-robocon`


## 📝 文件要求

### 必需文件
- `README.md`

### 可选文件
- `CHANGELOG.md`
- `CONTRIBUTORS.md`


## 🚀 快速开始

1. 在相应子目录下创建项目文件夹
2. 添加项目 README.md 文件
3. 按照命名规范组织文件结构
4. 更新上级目录的索引文件

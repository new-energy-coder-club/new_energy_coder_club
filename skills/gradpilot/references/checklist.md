# 留学申请材料检查表

> 按申请阶段使用，完成后打勾。

---

## 阶段一：准备期（申请季前 6-9 个月）

### 考试与成绩
- [ ] 确定目标项目所需考试（托福/雅思、GRE/GMAT、GRE Sub）
- [ ] 注册考试并制定复习计划
- [ ] 考出满意分数（建议预留重考时间）
- [ ] 确认成绩有效期覆盖入学时间
- [ ] 了解送分流程与代码（ETS / 雅思官方 / 学校官网）

### 背景提升
- [ ] 确认科研/实习/项目经历至少 2-3 段
- [ ] 如有论文，确认作者顺序与发表状态（已发表/在审/在写）
- [ ] 整理作品集 / GitHub / 个人网站（设计/CS 类推荐）

### 选校与信息
- [ ] 建立选校 Excel/Notion 表格（项目、Deadline、材料要求、状态）
- [ ] 确认每个项目的申请轮次与截止日期（注意时区！）
- [ ] 确认是否有特殊要求（Writing Sample、Video Essay、面试）

---

## 阶段二：材料准备期（申请季前 3-6 个月）

### 推荐信
- [ ] 确定 2-3 位推荐人（学术 + 职业组合）
- [ ] 与推荐人正式沟通，确认意愿与可用性
- [ ] 向推荐人提供材料包（CV、成绩单、申请项目清单、SOP 草稿、截止日期）
- [ ] 提醒推荐人 Deadline（提前 2 周、3 天分别提醒）

### 文书
- [ ] 起草 SOP / PS 初稿
- [ ] 起草 Personal History / Diversity Statement（如需）
- [ ] 修改 3 轮以上（自己→朋友→导师/中介）
- [ ] 最终校对（Grammarly + 人工朗读）
- [ ] 按项目定制版本（一校一版，忌群发）

### CV / Resume
- [ ] 整理中英文 CV
- [ ] 确认格式统一（字体、间距、时间倒序）
- [ ] 请他人审阅排版与内容

### 成绩单与认证
- [ ] 从学校教务处开具中英文成绩单
- [ ] 开具在读证明 / 毕业证学位证中英文版
- [ ] 办理 WES / 学信网认证（部分项目需要，耗时 4-8 周）
- [ ] 扫描所有文件为高清 PDF

---

## 阶段三：网申提交期（Deadline 前 1 个月）

### 网申系统
- [ ] 注册所有目标学校的网申账号
- [ ] 填写个人信息（与护照完全一致）
- [ ] 填写教育背景（学校官方英文名称）
- [ ] 填写工作经历与科研项目
- [ ] 上传文书（PDF，文件名规范：LastName_FirstName_SOP.pdf）
- [ ] 上传 CV
- [ ] 上传成绩单/认证报告
- [ ] 上传作品集/Writing Sample（如需）

### 送分与缴费
- [ ] 托福/雅思官方送分（提前 2-4 周）
- [ ] GRE/GMAT 官方送分
- [ ] 支付申请费（或申请 Fee Waiver）
- [ ] 保存缴费凭证与申请确认邮件

### 推荐信跟进
- [ ] 在系统中确认推荐人信息正确（姓名、邮箱）
- [ ] 确认推荐人收到网申系统邮件
- [ ] 提交前确认推荐信状态为 "Submitted"

---

## 阶段四：提交后（Deadline 后）

- [ ] 保存每个项目的申请确认页与申请号
- [ ] 检查申请状态页面是否齐全（绿勾 checklist）
- [ ] 补充遗漏材料（如发现缺件立即联系招生办）
- [ ] 准备面试（整理常见问题、模拟练习）
- [ ] 关注邮箱（含垃圾邮件文件夹）

---

## 阶段五：Offer 选择与入学准备

- [ ] 整理所有 Offer / Waitlist / Rejection
- [ ] 比较 Offer（Funding、导师、课程、地点、就业）
- [ ] 必要时请求延期 Deposit Deadline
- [ ] 交 Deposit 确认入学
- [ ] 申请 I-20 / CAS / COE（视国家而定）
- [ ] 预约签证面签
- [ ] 体检与疫苗接种
- [ ] 预订机票与安排住宿
- [ ] 加入新生群 / 联系学长学姐

# GradPilot · 留学申请全流程助手

> 专为硕/博研究生申请打造的系统化工作流，覆盖从选校到入学的全周期。

## 何时使用本 Skill

- 用户提及留学、申请、选校、文书、推荐信、CV、套磁、面试等关键词
- 需要制定申请时间线或检查材料清单
- 需要润色/修改 PS、SOP、CV 或推荐信
- 需要模拟面试或准备面试问题
- 需要评估个人背景与目标项目的匹配度

## 核心原则

1. **个性化优先**：不套用模板，基于用户背景（GPA、科研/实习、语言成绩、推荐信资源）给出针对性建议
2. **信息透明**：标注每一项建议的依据（项目官网、往年录取数据、领域常识）
3. **最小可行**：先完成再完美，避免过度打磨导致错过截止日期

## 1. 选校定位（School Selection）

### 1.1 信息收集清单

在给出选校建议前，先向用户确认以下信息：

| 维度 | 关键问题 |
|------|----------|
| **学位目标** | Master（就业导向/研究导向）或 PhD？是否考虑跳板硕？ |
| **专业方向** | 具体细分方向（如 CS→ML/Systems/HCI） |
| **地区偏好** | 美国、英国、欧洲大陆、加拿大、澳洲、港新？ |
| **资金需求** | 是否需要全奖/RA/TA？是否接受自费？ |
| **背景概况** | GPA（百分制/4.0/4.3）、托福/雅思、GRE/GMAT、科研/实习/论文 |
| **时间约束** | 申请季（如 25Fall）、是否已考出语言/GRE |
| **长期规划** | 回国就业、海外就业、继续读博、移民倾向？ |

### 1.2 选校梯度策略

- **冲刺校（Reach）**：1-2 所，录取率 <20%，需有明显亮点或彩票性质
- **匹配校（Match）**：2-3 所，背景与往年录取中位数相当
- **保底校（Safe）**：1-2 所，GPA/语言成绩显著高于往年平均，且愿意就读

### 1.3 项目调研维度

针对每个目标项目，建议整理：

1. **Admission Stats**：平均 GPA、语言分数、录取率、班级规模
2. **Curriculum**：必修课、选修课自由度、Capstone/Thesis 要求
3. **Faculty**：是否有研究方向匹配的老师（PhD 必需）
4. **Funding**：奖学金覆盖比例、RA/TA 机会、Co-op/Internship
5. **Career Outcome**：就业率、平均起薪、主要雇主、校友网络
6. **Location**：生活成本、气候、华人社区、实习/就业地理优势

> 推荐工具：一亩三分地、GradCafe、项目官网 FAQ / Class Profile、LinkedIn 校友搜索

## 2. 申请时间线（Timeline）

### 2.1 标准 25Fall 时间线（以美国 PhD 为例）

| 时间段 | 关键任务 |
|--------|----------|
| **3-5 月** | 确定方向、初步选校、考 GRE（如需）、联系推荐人 |
| **6-8 月** | 深度调研项目、套磁（PhD 优先）、起草 SOP/PS、考托福/雅思 |
| **9-10 月** | 修改文书定稿、确认推荐人提交意愿、填写网申系统、WES 认证（如需） |
| **11-12 月** | 提交申请（大部分 Deadline 为 12.1 / 12.15 / 1.1）、跟进推荐信 |
| **1-3 月** | 面试准备与参加面试、补充材料、等待结果 |
| **4-5 月** | 收 Offer、比较 Funding、参加 Open House、交 Deposit |
| **5-8 月** | 签证、体检、住宿、机票、行前准备 |

### 2.2 英联邦/港新时间线

- 英国：多数滚动录取，建议 10-11 月前提交；部分热门项目 1 月截止
- 港新：主轮通常在 11-12 月，部分有早鸟轮（9-10 月）
- 欧洲：分国家差异大，荷兰/瑞士/德国多为 1-4 月截止

## 3. 文书撰写（Application Documents）

### 3.1 文书核心逻辑

所有文书围绕一个核心：**“为什么是你 + 为什么是这个项目”**

- **动机（Motivation）**：什么经历/问题驱动你申请该方向？
- **能力（Capability）**：你具备什么知识/技能/经验来完成学业？
- **匹配（Fit）**：该项目/导师/实验室如何 specifically 帮助你实现目标？
- **规划（Plan）**：毕业后短期（3-5 年）与长期规划是什么？

### 3.2 PS vs SOP 区别

| 维度 | Personal Statement (PS) | Statement of Purpose (SOP) |
|------|------------------------|---------------------------|
| 侧重点 | 个人故事、成长经历、价值观 | 学术/职业目标、研究兴趣、技术背景 |
| 情感比例 | 40% 叙事 + 60% 专业 | 20% 叙事 + 80% 专业 |
| 常见要求 | 英国、部分美国硕士 | 美国 PhD、研究型硕士 |

### 3.3 CV / Resume 要点

- **长度**：PhD/学术型 CV 可 2 页；硕士申请 Resume 建议 1 页
- **模块**：Education → Research Experience → Publications → Work/Internship → Projects → Skills → Awards
- **量化**：用数字说明贡献（如 improved accuracy by 15%, served 200+ users）
- **格式**：PDF，简洁字体（Times New Roman / Calibri / Arial），避免花哨模板

### 3.4 推荐信策略

- **推荐人优先级**：熟悉度 > 职称 > 学校名气
- **理想组合**：
  - 科研导师（证明研究能力）
  - 课程教授（证明学术潜力）
  - 实习/工作主管（证明实践/团队能力）
- **沟通要点**：提供 CV、成绩单、申请项目清单、推荐信草稿素材（bullet points）
- **跟进**：Deadline 前 2 周提醒，前 3 天再次确认

## 4. 套磁与面试（Networking & Interview）

### 4.1 套磁邮件原则

- **时机**：PhD 建议 6-9 月；硕士视项目而定（研究型建议套磁）
- **对象**：AP/副教授优先（招生意愿强、回复率高），大教授可尝试但需精准匹配
- **邮件结构**：
  1. 标题：`Prospective PhD Student — [Your School] — [Research Area]`
  2. 自我介绍（1 句）
  3. 为何联系对方（具体提及 1-2 篇论文/项目）
  4. 你的相关背景与技能（2-3 句，量化）
  5. 明确诉求（询问招生名额/Funding/是否愿意进一步沟通）
  6. 附件：CV
- **禁忌**：群发模板、附件过大、同时套同一系多位老师

### 4.2 面试准备

**常见面试形式**：
- Behavioral（动机、职业规划、团队合作）
- Technical（专业知识、项目深挖、Coding/白板题）
- Research（PhD 必问：未来研究计划、读过的论文、感兴趣的问题）

**高频问题**：
1. Tell me about yourself / Walk me through your CV
2. Why this program? Why now?
3. What are your research interests? (PhD)
4. Tell me about a challenge you faced in a project
5. What do you want to do after graduation?
6. Do you have any questions for us?

**反问环节建议**：
- 实验室目前的项目方向与资金来源
- 毕业生的典型去向
- 导师的指导风格（hands-on vs hands-off）
- Funding 覆盖年限与续期条件

## 5. 网申与材料提交

### 5.1 网申系统 checklist

- [ ] 个人信息（姓名与护照完全一致）
- [ ] 教育背景（学校官方英文名称、GPA 换算说明）
- [ ] 成绩单（中英文，学校盖章/密封；部分需学信网/WES 认证）
- [ ] 语言/GRE 送分（提前 2-4 周操作，确认代码正确）
- [ ] 文书上传（严格遵循字数/页数/格式要求）
- [ ] 推荐信（确认推荐人收到链接并提交）
- [ ] 申请费（部分项目可 waive，需提前发邮件申请）
- [ ] 作品集/Writing Sample（设计/建筑/部分文科需要）

### 5.2 提交前 48 小时检查

1. 所有文件名规范：`LastName_FirstName_SOP.pdf`
2. PDF 可正常打开，无乱码
3. 文书无拼写/语法错误（可用 Grammarly / ChatGPT 辅助）
4. 推荐人系统显示 "Submitted"
5. 信用卡额度充足

## 6. Offer 选择与后续

### 6.1 Offer 比较维度

| 维度 | 权重建议 |
|------|----------|
| 导师/实验室（PhD） | 极高 |
| Funding 覆盖度 | 高 |
| 项目声誉与专排 | 高 |
| 地理位置与就业 | 中高 |
| 课程设置灵活性 | 中 |
| 生活成本 | 中 |
| 综合排名 | 中低 |

### 6.2 交 Deposit 前确认

- 是否有更好的 Offer 还在等待（可礼貌发邮件问学校能否延期 Deadline）
- Funding 细节（是否包含 Health Insurance、Stipend 发放频率）
- 是否有隐性条件（如 maintaining 3.5 GPA）

## 8. 常见问题排查

| 问题 | 建议 |
|------|------|
| GPA 偏低（<3.3/4.0） | 用高专业课 GPA、排名百分比、后期成绩上升趋势解释；用科研/实习弥补 |
| 转专业申请 | 强调先修课、自学路径、相关项目/科研；考虑桥梁项目 |
| 无科研经历申 PhD | 优先申 Master 转 PhD；或先积累 1-2 年 RA 经历 |
| 推荐人失联 | Deadline 前 1 周更换推荐人；提前准备备选 |
| 被 Waitlist | 发 Love Letter（更新成绩/奖项/新论文）；同时积极准备其他选项 |
| 全聚德 | 复盘文书与选校清单；Gap 一年积累经历；降低梯度重新申请 |

# 工商业光伏电站分层分级评分（两步走）

本 skill 把 `工商业光伏电站分层分级_两步走.xlsx` 模板里的两套评级方法固化成可重复执行的工具：

- **第一步 · 简版模式**：六维度直接评 A/B/C/D，木桶 / 多数 / 一票否决三规则联用，定出综合等级。
- **第二步 · 复杂版模式**：十分制加权评分（25/20/15/10/20/10），财务再拆 4 个子项（5a~5d），按等级折算分数后求和、加减分调整、再对照分数区间定级。

两步走共用同一组**加减分项**与**基准 IRR 区间**。

## 何时用

- 用户提供（或要求读取）一份光伏项目数据，希望自动给出评级与基准 IRR。
- 用户提到「工商业光伏 / 分层分级 / 两步走 / 评级 / 评审 / 投资判定 / IRR 基准 / 木桶原则」等关键词。
- 用户给出一个 `.xlsx` 文件想让你直接读出来跑评分（典型路径：`D:\OneDrive\Desktop\副本工商业光伏电站分层分级_两步走.xlsx`）。

## 工作流

### Step 0 · 拿到项目数据

只要满足下面任一条件即可开始评分：

1. 用户直接给出每个维度的等级（如「企业 A、省份 B、消纳 B、材料 A、财务 B、行业 B」）。
2. 用户给出原始指标（如年营收、年用电量、年发电量、自用比例、所在省份、行业、备案/接入材料状态等），由你**先映射成等级**再评分。
3. 用户给出一个 xlsx 路径，由 `scripts/parse_template.py` 读入按列填好的项目行。

如关键字段缺失（行业 / 财务 / 材料 / 省份 / 自用比例 / 企业性质中任意一项无法判定），先向用户**一次性**问清楚再评分，不要逐项追问。

### Step 1 · 简版评级（必出）

读取 `references/rating-rules.md` 里六维度的 A/B/C/D 划分基准，对照项目数据为每个维度定级，然后：

1. **木桶原则（推荐）**：综合等级 = 六维度最低等级。
2. **多数原则**：≥4 个维度同等级则定为该等级；分歧时以维度 5（财务）和维度 6（行业）为准。
3. **一票否决**：企业 D / 财务 D 且无强担保 / 材料 D（备案或接入缺失）/ 行业 D（房地产、P2P、教培）→ 直接定 D 或不投。

默认输出**木桶**的结果，并附**多数**结果作对照。

### Step 2 · 复杂版评分（如指标足够则直接出，否则提示）

调用 `scripts/score.py`，传入维度等级（含 5a/5b/5c/5d 四个子项）和加减分项。脚本完成：

1. 各维度等级 → 折算分数（A=100% / B=75% / C=50% / D=25%）。
2. 财务维度 = 5a×0.8 + 5b×0.5 + 5c×0.4 + 5d×0.3 的"权重内子分数"加总（其本身满分 2.0）。
3. 基础总分 = 6 个维度得分相加；最终总分 = 基础 + 加分 - 扣分（封顶 10、保底 0）。
4. 对照分数区间：A 8.0~10、B 6.0~7.9、C 4.0~5.9、D 0~3.9。
5. 输出基准 IRR 区间，并把加减分项的 IRR 调整也合并进去。

### Step 3 · 输出报告

最终报告至少包含：

- 项目名称、各维度等级与得分（含财务子项 5a/5b/5c/5d）。
- 基础总分、加减分明细、最终总分、综合等级。
- 基准 IRR 区间 → IRR 调整 → **最终 IRR 区间**。
- 对照木桶 / 多数 / 一票否决三种简版结论。
- 关键风险点（被打到 C/D 的维度逐条点名）与补救建议（材料补办、增信、配储能等）。

### Step 4 · 写回模板（用户明确要求"自动填"时）

调用 `scripts/fill_template.py`，把每个项目作为一行追加到 xlsx 的两个示例区下方。**默认输出副本** `<原文件>_filled.xlsx`，不覆盖原件。详见 `references/workflow.md`。

## 脚本

- `scripts/score.py`：评分主入口，CLI 与函数库双用。详见脚本头注释。
- `scripts/parse_template.py`：读取「两步走」模板示例区或一行项目数据。
- `scripts/fill_template.py`：把评分结果**写回** xlsx 模板（默认副本，不改原件）。

## 重要规则（必读）

1. **保留原表两步走逻辑**：简版必出，复杂版作为"进阶定量结果"，不要用复杂版分数反推简版等级。
2. **一票否决优先级最高**：触发后**直接 D 或不投**，不再走分数对照表。
3. **加减分顺序**：先算基础分，再加减，最后封顶/保底。
4. **IRR 区间不写死**：基础区间 + 调整百分点 = 最终区间。
5. **不替用户做决策**：评级结论给出后，明确说明"由投资评审会确认最终立项"。
